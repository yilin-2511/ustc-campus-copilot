# 中国科大"一〇七杯"算力与智能体开发大赛 - 比赛通知

> **发布日期**：2026年6月23日  
> **主办单位**：教务处、网络信息中心、学生工作部（处）、研究生院、对外联络与基金事务处、团委、中国科大教育基金会  
> **战略支持**：阿里巴巴集团及阿里巴巴公益基金会

---

## 一、赛道设置

大赛设**算力平台赛道**和**智能体赛道**。

### 1、算力平台赛道
- 由组织方提供建议题目，也可自主选题
- 基于学校本科生算力平台（https://107.ustc.edu.cn/）
- 围绕智能体开发、功能开发或学科专业领域的创新应用
- 作品形式：软件系统、工具或学科专业创新应用实例等
- 鼓励将已有基础的项目迁移至本科生算力平台

### 2、智能体赛道 ⭐
- **开放式自主命题**
- 围绕**智慧科大校园场景需求**
- 方向：教学赋能、科研支持、校园生活、服务管理
- 作品形式：软件系统 / 软硬件结合系统
- 鼓励结合大语言模型、多智能体协作等AI技术

### 模型资源
- 校内本地部署大模型：**DeepSeek-V4、GLM5.2** 等
- 每日保障赛事 **50亿 Token** 调用额度
- 每支队伍分发独立 API key
- 阿里巴巴提供账号、token 资源及技术支持

## 二、参赛对象

- 我校全体在校生
- 每队不超过 **4人**，至少包含 **1名本科生**
- **本科生组别**：全部为本科生（含2026届本科毕业生）
- **研究生组别**：有研究生参加
- 每队只能报名**一个赛道**

## 五、评审方式

围绕四个维度综合评审：
1. **创新性**
2. **实用性**
3. **技术难度**
4. **完成度**

**评审阶段：**
- **初评**：设计文档 + 演示视频 + 相关材料
- **终评**：现场演示答辩

优秀作品将推荐参加"挑战杯"等更高级别赛事。

1. 原创性作品，严禁抄袭、剽窃、一稿多投
