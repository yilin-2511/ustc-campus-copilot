**可访问性**：❌ 完全需要 CAS 统一身份认证登录  
**登录流程**：`ot.ustc.edu.cn → passport.ustc.edu.cn → id.ustc.edu.cn/cas/login`  
**接入方式**：必须 Playwright 模拟 CAS 登录

### 平台功能（推断）

- 学生社区：学习交流、经验分享
- 课程资料：课件、笔记、真题
- 讨论区：课程问答、学习讨论

### 技术实现

```python
from playwright.async_api import async_playwright

async def login_ot(username: str, password: str):
    """通过 CAS 登录蜗壳学社"""
    async with async_playwright() as p:
        browser = await p.chromium.launch(headless=True)
        page = await browser.new_page()
        
        # 访问蜗壳学社 → 自动重定向到 CAS 登录页
        await page.goto("https://ot.ustc.edu.cn/")
        
        # 填写 CAS 登录表单
        await page.fill("#username", username)
        await page.fill("#password", password)
        await page.click("#login-submit")
        
        await page.wait_for_load_state("networkidle")
        return page
```

## 接入策略总结

```
实测结论：科大平台均无公开 API，统一策略如下：

1. HTTP 解析（主力）⭐⭐⭐
   - 适用：教务公共查询、图书馆、学工在线、报修系统
   - 工具：httpx + BeautifulSoup
   - 难度：低，页面结构稳定
   - 频率：每 1-2 秒一次请求

2. Playwright 浏览器自动化（补充）⭐⭐
   - 适用：蜗壳学社（需 CAS 登录）、瀚海教学网论坛
   - 工具：Playwright
   - 难度：中，需处理 CAS 重定向和验证码

3. 静态知识库（兜底）⭐
   - 适用：食堂菜单（无网页版）、活动信息（分散在多处）
   - 方式：手动整理 + 定期更新
   - 难度：低，但需人工维护
```
